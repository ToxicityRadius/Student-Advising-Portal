# Student Advising Portal - Recent Updates (Since April 25, 2026)

This document highlights the major features, improvements, and fixes added to the Student Advising Portal over the past week.

## 🎓 Advising & Academic Features

* **Automated PDF Grade Imports**
  - **What's new:** Advisers can now directly upload official PDF grade checklists. The system automatically reads the PDF and imports the student's grades into their record.
  - **Why it matters:** This significantly reduces manual data entry time and prevents typing errors during advising sessions.

* **Interactive Study Plans**
  - **What's new:** Added a visual Study Plan Checklist that organizes a student's curriculum term-by-term.
  - **Why it matters:** Both students and advisers can easily see which courses have been taken, what is currently in progress, and what remains to graduate.

* **Prerequisite Override System**
  - **What's new:** A complete workflow for managing course prerequisites. Students can request exceptions, and advisers/admins can review, approve, or reject these requests directly through the portal.

* **Smarter Elective & Retake Management**
  - **What's new:** The system now automatically handles default elective tracks for the 2023 and 2025 curriculums. Advisers also have the ability to override a student's elective track if needed.
  - **Why it matters:** It prevents students from accidentally taking the wrong electives and accurately tracks students who need to retake a specific course.

## 🔒 Security & System Reliability

* **Improved Privacy in Activity Logs**
  - General users and advisers will no longer see sensitive "Super-Administrator" actions in the system's activity logs.

* **Strict Student ID Formatting**
  - The portal now strictly enforces the standard 7-digit student number format everywhere, preventing accidental duplicate accounts or typos.

* **Tighter Access Controls**
  - **What's new:** High-risk actions (like modifying the core curriculum or managing administrative accounts) are now strictly locked down to Super-Administrators only.
  - **Why it matters:** This prevents accidental changes to the foundation of the advising portal by unauthorized staff.

* **Behind-the-Scenes Stability**
  - Upgraded our automated testing processes to catch bugs before they reach the live site.
  - Hardened the database to prevent errors when upgrading or changing curriculum rules.
